# 📝 React To-Do List Application

A sleek and responsive To-Do List application built with **React.js**. This project showcases the use of functional components, state management with Hooks, and a clean user interface.

## ✨ Key Features
- **Add Tasks:** Quickly add new items to your list.
- **Task Management:** Mark tasks as completed or delete them when done.
- **Responsive Design:** Optimized for both mobile and desktop screens.
- **Persistent State:** (Optional: If you used LocalStorage) Saves your tasks even after refresh.

## 🛠️ Built With
* **React.js** - Frontend library
* **CSS3** - Custom styling and animations
* **JavaScript ** - Logic and functionality
